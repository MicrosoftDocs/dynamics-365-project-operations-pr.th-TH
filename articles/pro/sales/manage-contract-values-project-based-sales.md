# <a name="project-based-contract-lines-overview"></a>ภารวมของรายละเอียดการให้บริการตามสัญญาตามโครงการ

_**นำไปใช้กับ:** Project Operations สำหรับสถานการณ์ตามทรัพยากร/วัสดุที่ไม่ได้เก็บในคลัง การปรับใช้งานแบบ Lite - จัดการกับการออกใบแจ้งหนี้ชั่วคราว_

รายละเอียดการให้บริการตามสัญญาตามโครงการใน Dynamics 365 Project Operations ออกแบบมาเพื่อเก็บค่าการประมาณการและข้อตกลงการเรียกเก็บเงินสำหรับองค์ประกอบเฉพาะของงานโครงการในการมีส่วนร่วม โครงสร้างของรายละเอียดการให้บริการตามสัญญาตามโครงการจะขยายสำหรับการประมาณการโครงการและสถานการณ์การเรียกเก็บเงินด้วยแนวคิดต่อไปนี้:

- วิธีการเรียกเก็บเงิน
- การแม็ปโครงการและงาน
- คลาสธุรกรรมที่รวมไว้
- วงเงินที่ต้องไม่เกิน
- การตั้งค่าการคิดค่าธรรมเนียม
- ประมาณการโดยใช้รายละเอียดการให้บริการตามสัญญา
- ลูกค้าในรายละเอียดการให้บริการตามสัญญา

ตารางต่อไปนี้มีฟิลด์ในแท็บ **ทั่วไป** ของรายละเอียดการให้บริการตามสัญญาตามโครงการที่ช่วยตั้งค่าพื้นฐานสำหรับรายละเอียด การประมาณการพื้นฐาน และการเตรียมการเรียกเก็บเงินสำหรับงานตามโครงการ

| ฟิลด์ | รายละเอียด | ผลกระทบขั้นปลาย |
| --- | --- | --- |
| **ชื่อ** | ชื่อของรายละเอียดการให้บริการตามสัญญา ซึ่งระบุส่วนประกอบที่ไม่ต่อเนื่องของสัญญาที่อยู่ระหว่างการประมาณ สำหรับสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากค่าที่สอดคล้องกันของรายการใบเสนอราคาตามโครงการ | ชื่อนี้ถูกคัดลอกไปยังรายการใบแจ้งหนี้โครงการที่สร้างขึ้นจากรายละเอียดการให้บริการตามสัญญานี้เมื่อสร้างใบแจ้งหนี้ |
| **วิธีการเรียกเก็บเงิน** | บนสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา นี่คือชุดตัวเลือกที่แสดงถึงรูปแบบการทำสัญญาหลักสองแบบที่สนับสนุนโดย Project Operations:</br>- **ราคาคงที่**</br>- **เวลาและวัสดุ** | ตามวิธีการเรียกเก็บเงินของรายละเอียดการให้บริการตามสัญญาที่อ้างอิง ธุรกรรมจริงจะได้รับการประมวลผล หากรายละเอียดการให้บริการตามสัญญาที่อ้างอิงตามจริงมีวิธีการเรียกเก็บเงินตามเวลาและวัสดุ ระบบจะสร้างเรกคอร์ดต้นทุนและยอดขายที่ยังไม่เรียกเก็บ หากรายละเอียดการให้บริการตามสัญญาที่อ้างอิงตามจริงมีวิธีการเรียกเก็บเงินแบบราคาคงที่ ระบบจะสร้างเฉพาะต้นทุนจริงเท่านั้น |
| **Project** | ใช้ฟิลด์นี้เพื่อระบุโครงการที่จะใช้ในการส่งมอบงานในการมีส่วนร่วมนี้ | ค่านี้จะใช้ร่วมกับ **งานที่รวมไว้** และ **คลาสธุรกรรมที่รวม** เพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดรายการข้อมูลจริงหรือประมาณการ |
| **งานที่รวมไว้** | ระบุว่ารายละเอียดการให้บริการตามสัญญานี้รวมงานโครงการทั้งหมดสำหรับโครงการที่เลือกหรือเฉพาะบางส่วนของงานหรือไม่ นี่เป็นชุดตัวเลือกที่มีค่าที่เป็นไปได้ดังต่อไปนี้:</br>- **งานโครงการทั้งหมด**</br>- **งานโครงการที่เลือกเท่านั้น** ค่าว่างในฟิลด์นี้เท่ากับการเลือก **งานโครงการทั้งหมด** | ถ้า **งานที่เลือกเท่านั้น** ถูกเลือก คุณสามารถเลือกงานเฉพาะและเชื่อมโยงกับรายละเอียดการให้บริการตามสัญญานี้ในแท็บ **การตั้งค่าการเรียกเก็บเงินงาน** บนหน้า **โครงการ** ค่านี้ใช้ร่วมกับ **โครงการ** และคลาส **รวมธุรกรรม** เพื่อแก้ไขการอ้างอิงรายการสัญญาในเรกคอร์ดรายการจริงหรือรายการโดยประมาณ |
| **รวมเวลา** | ค่า **ใช่**/**ไม่** ระบุว่าธุรกรรมเวลาหรือต้นทุนแรงงานในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าธุรกรรมเวลาหรือต้นทุนแรงงานจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่านี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดข้อมูลจริงหรือประมาณการ |
| **รวมค่าใช้จ่าย** | ค่า **ใช่**/**ไม่** ระบุว่าต้นทุนค่าใช้จ่ายในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าต้นทุนค่าใช้จ่ายจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่านี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดข้อมูลจริงหรือประมาณการ |
| **รวมวัสดุ** | ค่า **ใช่**/**ไม่** ระบุว่าต้นทุนวัสดุในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าต้นทุนวัสดุจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่านี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดข้อมูลจริงหรือประมาณการ |
| **รวมค่าธรรมเนียม** | ค่า **ใช่**/**ไม่** ระบุว่าต้นทุนค่าธรรมเนียมในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าค่าธรรมเนียมจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่านี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดข้อมูลจริงหรือประมาณการ |
| **จำนวนเงินตามสัญญา** | ในรายละเอียดการให้บริการตามสัญญาราคาคงที่ จำนวนนี้คือมูลค่าที่ตกลงกันซึ่งจะออกใบแจ้งหนี้ให้กับลูกค้าสำหรับส่วนประกอบงานทั้งหมดที่เกี่ยวข้องกับรายละเอียดการให้บริการตามสัญญานี้ ในรายละเอียดการให้บริการตามสัญญาเวลาและวัสดุ จำนวนนี้คือค่าประมาณที่จะออกใบแจ้งหนี้ให้กับลูกค้าสำหรับส่วนประกอบงานทั้งหมดที่เกี่ยวข้องกับรายละเอียดการให้บริการตามสัญญานี้ บนสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา เมื่อรายละเอียดการให้บริการตามสัญญาตามโครงการมีรายละเอียดรายการ ฟิลด์นี้จะถูกล็อกสำหรับการแก้ไขและสรุปจากจำนวนเงินในรายละเอียดการให้บริการตามสัญญา | เมื่อรายละเอียดการให้บริการตามสัญญามีรายละเอียดของรายการ ค่านี้สามารถแก้ไขได้โดยการเปลี่ยนจำนวนเงินในรายละเอียดรายการ ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างจำนวนเงินก่อนหักภาษีสำหรับเหตุการณ์การเรียกเก็บเงินตามงวด |
| **ภาษีที่ประเมิน** | ผู้ใช้สามารถแก้ไขฟิลด์นี้เพื่อป้อนจำนวนภาษีโดยประมาณในรายละเอียดการให้บริการตามสัญญา เมื่อรายละเอียดการให้บริการตามสัญญาตามโครงการมีรายละเอียดรายการ ฟิลด์นี้จะถูกล็อกสำหรับการแก้ไขและสรุปจากจำนวนเงินภาษีในรายละเอียดการให้บริการตามสัญญา | เมื่อรายละเอียดการให้บริการตามสัญญามีรายละเอียดของรายการ ค่านี้สามารถแก้ไขได้โดยการเปลี่ยนจำนวนเงินภาษีในรายละเอียดรายการ ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างภาษีสำหรับเหตุการณ์การเรียกเก็บเงินตามงวด |
| **จำนวนเงินตามสัญญาหลังหักภาษี** | จำนวนเงินตามรายละเอียดการให้บริการตามสัญญาหลังหักภาษี ฟิลด์นี้เป็นแบบอ่านอย่างเดียวและคำนวณเป็น **จำนวนเงินตามสัญญา + ภาษี** | ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างเหตุการณ์การเรียกเก็บเงินตามงวด |
| **วงเงินที่ต้องไม่เกิน** | ผู้ใช้สามารถแก้ไขฟิลด์นี้และใช้ได้เฉพาะในรายละเอียดการให้บริการตามสัญญาตามโครงการที่มีวิธีการเรียกเก็บเงินตั้งค่าเป็นเวลาและวัสดุ | ผู้ใช้สามารถแก้ไขฟิลด์นี้ เมื่อบ้อมูลจริงสำหรับเวลาหรือวัสดุอ้างถึงรายละเอียดการให้บริการตามสัญญาสำหรับเวลาและวัสดุ จำนวนเงินตามจริงจะถูกประเมินเทียบกับไม่เกินขีดจำกัด ในรายละเอียดการให้บริการตามสัญญานี้ การประเมินนี้จะเสร็จสมบูรณ์หลังจากที่มีการคำนวณยอดเงินที่รับผิดชอบอและยืนยันแล้ว |
| **งบประมาณของลูกค้า** | ค่านี้สามารถแก้ไขได้และถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา หากสัญญาสร้างจากใบเสนอราคา | ฟิลด์นี้ใช้สำหรับข้อมูลเท่านั้นและไม่มีความสำคัญดาวน์สตรีม |

## <a name="validation-rules-for-the-options-on-the-general-tab-of-project-based-contract-lines"></a>กฎการตรวจสอบความถูกต้องสำหรับตัวเลือกบนแท็บทั่วไปของรายละเอียดการให้บริการตามสัญญาตามโครงการ

กฎข้อที่ 1: หากฟิลด์ **รวมงาน** ว่างหรือตั้งค่าเป็น **งานโครงการทั้งหมด** งานทั้งหมดของโครงการจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญา

กฎข้อ 2: เมื่อฟิลด์ **รวมงาน** ว่างเปล่าหรือตั้งค่าเป็น **งานโครงการทั้งหมด** โครงการและคลาสธุรกรรมบางรายการสามารถรวมอยู่ในรายละเอียดการให้บริการตามสัญญาตามโครงการของสัญญาได้เพียงรายการเดียว

กฎข้อ 3: เมื่อฟิลด์ **รวมงาน** ตั้งค่าเป็น **งานโครงการที่เลือกเท่านั้น** โครงการและคลาสธุรกรรมบางรายการสามารถรวมอยู่ในรายละเอียดการให้บริการตามสัญญาตามโครงการของสัญญาได้หลายรายการ
