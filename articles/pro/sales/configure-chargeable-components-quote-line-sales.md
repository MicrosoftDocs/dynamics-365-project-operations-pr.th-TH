# <a name="configure-the-chargeable-components-of-a-quote-line---lite"></a>กำหนดค่าส่วนประกอบที่เรียกเก็บเงินของรายการใบเสนอราคา - Lite

_**นำไปใช้กับ:** การปรับใช้งานแบบ Lite - จัดการกับการออกใบแจ้งหนี้ชั่วคราว_

บรรทัดใบเสนอราคาตามโครงการมีแนวคิดส่วนประกอบ *รวม* และส่วนประกอบ *คิดค่าธรรมเนียมได้*

ส่วนประกอบที่รวมอยู่ภายใต้:

  - วิธีการเรียกเก็บเงินและการแยกลูกค้า
  - วงเงินที่ต้องไม่เกิน 
  - การตั้งค่าความถี่ของใบแจ้งหนี้ที่กำหนดไว้ในบรรทัดใบเสนอราคาตามโครงการ

ชุดย่อยของส่วนประกอบที่รวมอยู่สามารถทำเครื่องหมายเป็นคิดค่าธรรมเนียมได้โดยใช้ฟิลด์ **ประเภทการเรียกเก็บเงิน** ฟิลด์ **ประเภทการเรียกเก็บเงิน** คือชุดตัวเลือกที่อนุญาตให้ใช้ค่าต่อไปนี้ในบริบทของบรรทัดใบเสนอราคา:

  - คิดค่าธรรมเนียมได้
  - คิดค่าธรรมเนียมไม่ได้

ส่วนประกอบที่สามารถคิดค่าธรรมเนียมได้สามารถกำหนดได้ตามงาน บทบาท และประเภทธุรกรรม

การคิดค่าธรรมเนียมถูกกำหนดไว้ในงานสำหรับบรรทัดใบเสนอราคาและใช้กับคลาสธุรกรรมทั้งหมดที่รวมอยู่ในรายการนั้น ถ้าฟิลด์ **รวมงาน** ถูกตั้งค่าเป็น **ทั้งโครงการ** หรือปล่อยว่าง **งานที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้

การคิดค่าธรรมเนียมที่กำหนดไว้สำหรับบรรทัดใบเสนอราคาและจะใช้กับคลาสธุรกรรม **เวลา** เท่านั้น ถ้าฟิลด์ **รวมเวลา** ถูกตั้งค่าเป็น **ไม่** บนบรรทัดใบเสนอราคาของโครงการ แท็บ **บทบาทที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้

การคิดค่าธรรมเนียมที่กำหนดไว้บนประเภทธุรกรรมสำหรับบรรทัดใบเสนอราคาและจะใช้กับคลาสธุรกรรม **ค่าใช้จ่าย** เท่านั้น ถ้าฟิลด์ **รวมค่าใช้จ่าย** ถูกตั้งค่าเป็น **ไม่** บนบรรทัดใบเสนอราคาของโครงการ แท็บ **ประเภทที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a>อัปเดตงานโครงการเป็นแบบคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้

งานโครงการสามารถคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้ในบริบทของบรรทัดใบเสนอราคาตามโครงการเฉพาะ ซึ่งทำให้การตั้งค่าต่อไปนี้เป็นไปได้:

หากมีบรรทัดใบเสนอราคาตามโครงการ **เวลา** และงาน **T1** งานนี้เชื่อมโยงกับบรรทัดใบเสนอราคาที่คิดค่าธรรมเนียมได้ หากมีบรรทัดใบเสนอราคาที่สองซึ่งรวมถึง **ค่าใช้จ่าย** คุณสามารถเชื่อมโยงงาน **T1** บนบรรทัดใบเสนอราคาเป็นแบบไม่คิดค่าธรรมเนียม ผลที่ได้คือเวลาทั้งหมดที่บันทึกไว้ในงานนั้นสามารถคิดค่าธรรมเนียมได้และค่าใช้จ่ายทั้งหมดจะบันทึกบนงานเป็นไม่สามารถคิดค่าธรรมเนียมได้

ชนิดการเรียกเก็บเงินของงานสามารถกำหนดค่าได้ในแท็บ **งานที่สามารถคิดค่าธรรมเนียมได้** ของรายการใบเสนอราคาตามโครงการโดยการอัปเดตฟิลด์ **ชนิดการเรียกเก็บเงิน** บนตารางย่อย **งานในรายการใบเสนอราคา** หรือคุณสามารถอัปเดตชนิดการเรียกเก็บเงินสำหรับงานโครงการในฟิลด์ **ชนิดการเรียกเก็บเงิน** บนตารางย่อยในการตั้งค่าการเรียกเก็บเงินงานของโครงการที่แสดงรายการใบเสนอราคาที่เกี่ยวข้องกับงาน

### <a name="update-a-role-to-be-chargeable-or-non-chargeable"></a>อัปเดตบทบาทเป็นแบบคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้

บทบาทสามารถคิดค่าธรรมเนียมได้หรือไม่ได้ในบริบทของบรรทัดใบเสนอราคาตามโครงการเฉพาะ

ชนิดการเรียกเก็บเงินของบทบาทสามารถกำหนดค่าได้ในแท็บ **บทบาทที่สามารถคิดค่าธรรมเนียมได้** ของรายการใบเสนอราคาโดยการอัปเดตฟิลด์ **ชนิดการเรียกเก็บเงิน** บนตารางย่อย **บทบาทที่สามารถคิดค่าธรรมเนียมได้**

### <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a>อัปเดตประเภทธุรกรรมเป็นแบบคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้

ประเภทธุรกรรมสามารถเป็นแบบคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้ในบรรทัดใบเสนอราคา

ชนิดการเรียกเก็บเงินของธุรกรรมสามารถกำหนดค่าได้ในแท็บ **ประเภทที่คิดค่าธรรมเนียมได้** ของรายการใบเสนอราคาโดยการอัปเดตฟิลด์ **ชนิดการเรียกเก็บเงิน** บนตารางย่อย **ประเภทที่คิดค่าธรรมเนียมได้**
