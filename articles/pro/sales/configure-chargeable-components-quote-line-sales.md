<span data-ttu-id="321d7-119">การคิดค่าธรรมเนียมที่กำหนดไว้บนประเภทธุรกรรมสำหรับบรรทัดใบเสนอราคาและจะใช้กับคลาสธุรกรรม **ค่าใช้จ่าย** เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="321d7-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="321d7-120">ถ้าฟิลด์ **รวมค่าใช้จ่าย** ถูกตั้งค่าเป็น **ไม่** บนบรรทัดใบเสนอราคาของโครงการ แท็บ **ประเภทที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้</span><span class="sxs-lookup"><span data-stu-id="321d7-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

<span data-ttu-id="321d7-122">งานโครงการสามารถคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้ในบริบทของบรรทัดใบเสนอราคาตามโครงการเฉพาะ ซึ่งทำให้การตั้งค่าต่อไปนี้เป็นไปได้:</span><span class="sxs-lookup"><span data-stu-id="321d7-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="321d7-123">หากมีบรรทัดใบเสนอราคาตามโครงการ **เวลา** และงาน **T1** งานนี้เชื่อมโยงกับบรรทัดใบเสนอราคาที่คิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="321d7-123">If a project-based quote line includes **Time** and the task **T1**, the task is associated to the quote line as chargeable.</span></span> <span data-ttu-id="321d7-124">หากมีบรรทัดใบเสนอราคาที่สองซึ่งรวมถึง **ค่าใช้จ่าย** คุณสามารถเชื่อมโยงงาน **T1** บนบรรทัดใบเสนอราคาเป็นแบบไม่คิดค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="321d7-124">If there is a second quote line that includes **Expenses**, you can associate the **T1** task on the quote line as non-chargeable.</span></span> <span data-ttu-id="321d7-125">ผลที่ได้คือเวลาทั้งหมดที่บันทึกไว้ในงานนั้นสามารถคิดค่าธรรมเนียมได้และค่าใช้จ่ายทั้งหมดจะบันทึกบนงานเป็นไม่สามารถคิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="321d7-125">The result is that all time recorded on the task is chargeable and all expenses recorded on the task are non-chargeable.</span></span>
