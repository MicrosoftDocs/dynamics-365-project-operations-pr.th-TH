<span data-ttu-id="13bf2-106">ส่วนประกอบที่รวมอยู่ภายใต้:</span><span class="sxs-lookup"><span data-stu-id="13bf2-106">Included components are subject to:</span></span>

  - <span data-ttu-id="13bf2-107">วิธีการเรียกเก็บเงินและการแยกลูกค้า</span><span class="sxs-lookup"><span data-stu-id="13bf2-107">Billing method and customer splits</span></span>

  - <span data-ttu-id="13bf2-109">การตั้งค่าความถี่ของใบแจ้งหนี้ที่กำหนดไว้ในบรรทัดใบเสนอราคาตามโครงการ</span><span class="sxs-lookup"><span data-stu-id="13bf2-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="13bf2-110">ชุดย่อยของส่วนประกอบที่รวมอยู่สามารถทำเครื่องหมายเป็นคิดค่าธรรมเนียมได้โดยใช้ฟิลด์ **ประเภทการเรียกเก็บเงิน**</span><span class="sxs-lookup"><span data-stu-id="13bf2-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="13bf2-111">ฟิลด์ **ประเภทการเรียกเก็บเงิน** คือชุดตัวเลือกที่อนุญาตให้ใช้ค่าต่อไปนี้ในบริบทของบรรทัดใบเสนอราคา:</span><span class="sxs-lookup"><span data-stu-id="13bf2-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

<span data-ttu-id="13bf2-114">ส่วนประกอบที่สามารถคิดค่าธรรมเนียมได้สามารถกำหนดได้ตามงาน บทบาท และประเภทธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="13bf2-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="13bf2-115">การคิดค่าธรรมเนียมถูกกำหนดไว้ในงานสำหรับบรรทัดใบเสนอราคาและใช้กับคลาสธุรกรรมทั้งหมดที่รวมอยู่ในรายการนั้น</span><span class="sxs-lookup"><span data-stu-id="13bf2-115">Chargeability is defined on the tasks for a quote line and applies to all transaction classes included on that line.</span></span> <span data-ttu-id="13bf2-116">ถ้าฟิลด์ **รวมงาน** ถูกตั้งค่าเป็น **ทั้งโครงการ** หรือปล่อยว่าง **งานที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้</span><span class="sxs-lookup"><span data-stu-id="13bf2-116">If the **Include Tasks** field is set to **Entire project** or left blank, the **Chargeable Tasks** tab isn't available.</span></span>

<span data-ttu-id="13bf2-119">การคิดค่าธรรมเนียมที่กำหนดไว้บนประเภทธุรกรรมสำหรับบรรทัดใบเสนอราคาและจะใช้กับคลาสธุรกรรม **ค่าใช้จ่าย** เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="13bf2-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="13bf2-120">ถ้าฟิลด์ **รวมค่าใช้จ่าย** ถูกตั้งค่าเป็น **ไม่** บนบรรทัดใบเสนอราคาของโครงการ แท็บ **ประเภทที่คิดค่าธรรมเนียมได้** จะไม่สามารถใช้งานได้</span><span class="sxs-lookup"><span data-stu-id="13bf2-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="13bf2-121">อัปเดตงานโครงการเป็นแบบคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้</span><span class="sxs-lookup"><span data-stu-id="13bf2-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="13bf2-122">งานโครงการสามารถคิดค่าธรรมเนียมได้หรือคิดค่าธรรมเนียมไม่ได้ในบริบทของบรรทัดใบเสนอราคาตามโครงการเฉพาะ ซึ่งทำให้การตั้งค่าต่อไปนี้เป็นไปได้:</span><span class="sxs-lookup"><span data-stu-id="13bf2-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="13bf2-123">หากมีบรรทัดใบเสนอราคาตามโครงการ **เวลา** และงาน **T1** งานนี้เชื่อมโยงกับบรรทัดใบเสนอราคาที่คิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="13bf2-123">If a project-based quote line includes **Time** and the task **T1**, the task is associated to the quote line as chargeable.</span></span> <span data-ttu-id="13bf2-124">หากมีบรรทัดใบเสนอราคาที่สองซึ่งรวมถึง **ค่าใช้จ่าย** คุณสามารถเชื่อมโยงงาน **T1** บนบรรทัดใบเสนอราคาเป็นแบบไม่คิดค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="13bf2-124">If there is a second quote line that includes **Expenses**, you can associate the **T1** task on the quote line as non-chargeable.</span></span> <span data-ttu-id="13bf2-125">ผลที่ได้คือเวลาทั้งหมดที่บันทึกไว้ในงานนั้นสามารถคิดค่าธรรมเนียมได้และค่าใช้จ่ายทั้งหมดจะบันทึกบนงานเป็นไม่สามารถคิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="13bf2-125">The result is that all time recorded on the task is chargeable and all expenses recorded on the task are non-chargeable.</span></span>
