# <a name="manage-a-proforma-invoice---lite"></a>จัดการใบแจ้งหนี้ชั่วคราว - Lite

_**นำไปใช้กับ:** การปรับใช้งานแบบ Lite - จัดการกับการออกใบแจ้งหนี้ชั่วคราว_

ใน Dynamics 365 Project Operations ใบแจ้งหนี้ชั่วคราวถูกสร้างขึ้นเพื่อเป็นส่วนขยายของใบแจ้งหนี้ใน Dynamics 365 Sales อย่างไรก็ตาม มีความแตกต่างหลายประการในกระบวนการออกใบแจ้งหนี้ระหว่าง Sales และ Project Operations เมื่อพูดถึงการออกใบแจ้งหนี้ ตัวอย่างเช่น ไม่สามารถสร้างใบแจ้งหนี้ใหม่จากหน้า **รายการใบแจ้งหนี้** ใน Project Operations แต่สามารถทำได้ใน Sales ความแตกต่างและส่วนขยายเหล่านี้มีไว้เพื่อรองรับกระบวนการออกใบแจ้งหนี้สำหรับโครงการที่แตกต่างจากใบแจ้งหนี้ทั่วไปสำหรับใบสั่งขาย

> [!IMPORTANT]
> เนื่องจากความแตกต่าง อย่าใช้ใบแจ้งหนี้ใน Sales และ Project Operations แทนกัน

## <a name="invoice-line-details"></a>รายละเอียดรายการใบแจ้งหนี้

แต่ละรายการใบแจ้งหนี้ในใบแจ้งหนี้โครงการประกอบด้วยรายละเอียดรายการใบแจ้งหนี้ รายละเอียดรายการเหล่านี้เกี่ยวข้องกับยอดขายจริงที่ยังไม่เรียกเก็บเงินและเหตุการณ์สำคัญที่เกี่ยวข้องกับรายละเอียดการให้บริการตามสัญญาที่อ้างอิงโดยรายการใบแจ้งหนี้ ธุรกรรมทั้งหมดนี้มีการทำเครื่องหมาย **พร้อมที่จะออกใบแจ้งหนี้**

สำหรับรายการ **ใบแจ้งหนี้เวลาและวัสดุ** รายละเอียดรายการใบแจ้งหนี้จะถูกจัดกลุ่มเป็น **คิดค่าธรรมเนียมได้** **คิดค่าธรรมเนียมไม่ได้** และ **ส่วนเสริม** บนหน้า **รายการใบแจ้งหนี้** รายละเอียด **รายการใบแจ้งหนี้ที่คิดค่าธรรมเนียมได้** จะรวมอยู่ในยอดรวมรายการใบแจ้งหนี้ **ส่วนเสริม** และ **จำนวนเงินตามจริงที่คิดค่าธรรมเนียมไม่ได้** ไม่รวมยอดรวมในรายการใบแจ้งหนี้

สำหรับรายการ **ใบแจ้งหนี้ราคาคงที่** รายละเอียดรายการใบแจ้งหนี้สร้างขึ้นจากเหตุการณ์สำคัญที่ทำเครื่องหมายเป็น **พร้อมออกใบแจ้งหนี้** ในรายละเอียดการให้บริการตามสัญญาที่เกี่ยวข้อง หลังจากสร้างรายละเอียดรายการใบแจ้งหนี้จากเหตุการณ์สำคัญแล้ว สถานะการเรียกเก็บเงินในเหตุการณ์สำคัญจะอัปเดตเป็น **สร้างใบแจ้งหนี้ของลูกค้าแล้ว**

### <a name="edit-invoice-line-details"></a>แก้ไขรายละเอียดบรรทัดใบแจ้งหนี้

ฟิลด์ต่อไปนี้มีอยู่ในรายละเอียดรายการใบแจ้งหนี้ที่ได้รับการสนับสนุนโดยยอดขายจริงที่ยังไม่เรียกเก็บเงิน

| เขตข้อมูล | รายละเอียด | ผลกระทบขั้นปลาย |
| --- | --- | --- |

รายการใบแจ้งหนี้เวลาและวัสดุ คุณสามารถสร้างรายละเอียดรายการใบแจ้งหนี้ใหม่ได้ รายละเอียดรายการใบแจ้งหนี้เหล่านี้ไม่ได้รับการสนับสนุนจากข้อมูลจริง ในรายการใบแจ้งหนี้ของรายการใบแจ้งหนี้เวลาและวัสดุ คุณสามารถเลือก **สร้าง** เพื่อสร้างรายละเอียดรายการใบแจ้งหนี้ใหม่สำหรับคลาสธุรกรรมที่รวมอยู่ในรายละเอียดการให้บริการตามสัญญา

## <a name="refresh-invoice-transactions"></a>รีเฟรชธุรกรรมของใบแจ้งหนี้

หากคุณมีข้อมูลจริงที่เข้ามาหลังจากสร้างใบแจ้งหนี้แล้ว คุณสามารถรวมข้อมูลจริงเหล่านี้ไว้ในใบแจ้งหนี้ได้

1. ใน **มุมมองรายการงานค้างของการเรียกเก็บเงิน** ทำเครื่องหมายข้อมูลเป็น **พร้อมที่จะออกใบแจ้งหนี้**   
2. เปิดแบบร่างใบแจ้งหนี้ชั่วคราว และบน **การดำเนินการ** Ribbon คลิก **รีเฟรชธุรกรรมรายการใบแจ้งหนี้**

  สิ่งนี้จะสร้างรายละเอียดรายการใบแจ้งหนี้สำหรับข้อมูลจริงที่เกินวันที่และทำเครื่องหมายเป็น **พร้อมที่จะออกใบแจ้งหนี้**; แต่ไม่รวมอยู่ในใบแจ้งหนี้

## <a name="product-based-invoice-lines"></a>รายละเอียดการให้บริการตามใบแจ้งหนี้ตามผลิตภัณฑ์

ใน Project Operations คุณสามารถสร้างรายการใบแจ้งหนี้สำหรับผลิตภัณฑ์ที่ไม่ได้ใช้กับโครงการใด ๆ หรือสำหรับโครงการทั้งหมดพร้อมกับรายการใบแจ้งหนี้ตามโครงการ รายการใบแจ้งหนี้เหล่านี้ถูกสร้างขึ้นเป็นรายละเอียดการให้บริการตามสัญญาตามผลิตภัณฑ์ และหลังจากทำเครื่องหมายว่าพร้อมที่จะออกใบแจ้งหนี้แล้ว รายการเหล่านี้จะถูกเพิ่มเป็นรายการใบแจ้งหนี้ตามผลิตภัณฑ์

หลังจากที่คุณเพิ่มรายการใบแจ้งหนี้ตามผลิตภัณฑ์แล้ว จะไม่สามารถเปลี่ยนแปลงได้ อย่างไรก็ตามสามารถลบออกจากร่างใบแจ้งหนี้ชั่วคราวได้
