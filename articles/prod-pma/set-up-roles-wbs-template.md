[!include [banner](../includes/banner.md)]

<span data-ttu-id="f5382-104">ผู้จัดการโครงการสามารถตั้งค่าเทมเพลตโครงสร้างการแบ่งงาน (WBS) ที่พวกเขาสามารถนำไปใช้ เมื่อพวกเขาสร้าง WBS สำหรับโครงการใหม่</span><span class="sxs-lookup"><span data-stu-id="f5382-104">Project managers can set up Work breakdown structure (WBS) templates that they can apply when they create a WBS for new projects.</span></span> <span data-ttu-id="f5382-105">ผู้จัดการโครงการสามารถเพิ่มบทบาท เมื่อพวกเขาสร้างแม่แบบ</span><span class="sxs-lookup"><span data-stu-id="f5382-105">Project managers can add roles when they create a template.</span></span> <span data-ttu-id="f5382-106">ใช้กระบวนงานต่อไปนี้เพื่อมอบหมายบทบาทให้กับแม่แบบ WBS</span><span class="sxs-lookup"><span data-stu-id="f5382-106">Use the following procedure to assign a role to a WBS template.</span></span>

1. <span data-ttu-id="f5382-107">เลือก **การจัดการโครงการและการบัญชี** > **การตั้งค่า** > **โครงการ** > **แม่แบบโครงสร้างการแบ่งงาน**</span><span class="sxs-lookup"><span data-stu-id="f5382-107">Select **Project management and accounting** > **Setup** > **Projects** > **Work breakdown structure templates**.</span></span>
