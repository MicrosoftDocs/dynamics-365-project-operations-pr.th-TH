3. <span data-ttu-id="bacb3-163">ระบุรหัสบรรทัดต้นทุนสำหรับธุรกรรมแต่ละชนิด</span><span class="sxs-lookup"><span data-stu-id="bacb3-163">Provide cost line ID for each transaction type.</span></span>
4. <span data-ttu-id="bacb3-164">เลือกวิธีการทำให้เสร็จสมบูรณ์เริ่มต้น:</span><span class="sxs-lookup"><span data-stu-id="bacb3-164">Select a default completion method:</span></span>

  - <span data-ttu-id="bacb3-167">**ด้วยตนเอง**: เปอร์เซ็นต์ของความสำเร็จจะถูกคำนวณโดยอัตโนมัติในโครงการ</span><span class="sxs-lookup"><span data-stu-id="bacb3-167">**Manual**: Percentage of completion is calculated automatically on a project.</span></span> <span data-ttu-id="bacb3-168">ค่าผลลัพธ์สามารถเปลี่ยนแปลงได้</span><span class="sxs-lookup"><span data-stu-id="bacb3-168">The resulting value can be changed.</span></span>

  > [!NOTE]
  > <span data-ttu-id="bacb3-169">สำหรับการคำนวณด้วยตนเอง เปอร์เซ็นต์ของความสำเร็จจะยังคงอยู่ใน **การคำนวณด้วยตนเอง** บนแท็บ **ทั่วไป** บนหน้า **การประมาณการ**</span><span class="sxs-lookup"><span data-stu-id="bacb3-169">For manual calculations, the percentage of completion is maintained in **Manual calculation** on the **General** tab on the **Estimate** page.</span></span>

5. <span data-ttu-id="bacb3-170">ใน **ความเสร็จสมบูรณ์ขึ้นอยู่กับ** เลือกค่าใดค่าหนึ่งต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="bacb3-170">In **Completion based on**, select one of the following values:</span></span>

  - <span data-ttu-id="bacb3-171">**จำนวน**: จำนวนในสกุลเงินทางบัญชีคำนวณเปอร์เซ็นต์ของความสำเร็จ</span><span class="sxs-lookup"><span data-stu-id="bacb3-171">**Amount**: The amount in accounting currency calculates the percentage of completion.</span></span>
  - <span data-ttu-id="bacb3-172">**หน่วย**: ปริมาณจะคำนวณเปอร์เซ็นต์ของความสำเร็จ</span><span class="sxs-lookup"><span data-stu-id="bacb3-172">**Unit**: The quantity calculates the percentage of completion.</span></span>
  - <span data-ttu-id="bacb3-173">**เส้นตรง**: ระบบจะคำนวณเปอร์เซ็นต์ของความสำเร็จตามสัดส่วนโดยใช้วันที่เริ่มต้นและวันที่สิ้นสุดโครงการเพื่อกำหนดความยาวของโครงการ</span><span class="sxs-lookup"><span data-stu-id="bacb3-173">**Straight line**: The system calculates the percentage of completion on a proportionate basis by using the project start and end dates to determine the length of the project.</span></span>

6. <span data-ttu-id="bacb3-174">เลือก **บรรทัดต้นทุน** เพื่อตรวจสอบรายการต้นทุนของแม่แบบต้นทุน</span><span class="sxs-lookup"><span data-stu-id="bacb3-174">Select **Cost lines** to review the cost lines of the cost template.</span></span> <span data-ttu-id="bacb3-175">ต้องมีรายการต้นทุนอย่างน้อยหนึ่งรายการสำหรับทุกชนิดธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="bacb3-175">At least one cost line is required for every transaction type.</span></span> <span data-ttu-id="bacb3-176">อย่างไรก็ตาม คุณสามารถสร้างรายการต้นทุนหลายรายการสำหรับชนิดธุรกรรมเดียวกัน และตั้งค่าแอตทริบิวต์ที่ต้องการสำหรับรายการเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="bacb3-176">However, you can create multiple cost lines for the same transaction types and set the desired attributes for these lines.</span></span>
7. <span data-ttu-id="bacb3-177">บนแท็บ **ประเภท** เลือกประเภทโครงการที่จะรวมไว้ในบรรทัดแม่แบบต้นทุน</span><span class="sxs-lookup"><span data-stu-id="bacb3-177">On the **Categories** tab, select the project categories to be included on the cost template line.</span></span>
8. <span data-ttu-id="bacb3-178">บนแท็บ **ทั่วไป** เลือกว่าจะรวมบรรทัดนี้ในเปอร์เซ็นต์ของการคำนวณความสำเร็จหรือไม่</span><span class="sxs-lookup"><span data-stu-id="bacb3-178">On the **General** tab, select if this line will be included in the percentage of completion calculation.</span></span>
9. <span data-ttu-id="bacb3-179">เลือกวิธีต้นทุนเพื่อทำให้เสร็จสมบูรณ์เพื่อใช้ในการคำนวณเปอร์เซ็นต์ของความสำเร็จ</span><span class="sxs-lookup"><span data-stu-id="bacb3-179">Select the cost to complete method to be used when calculating the percentage of completion.</span></span>
