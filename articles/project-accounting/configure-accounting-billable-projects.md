<span data-ttu-id="21707-105">Dynamics 365 Project Operations รองรับตัวเลือกการบัญชีที่หลากหลายสำหรับโครงการที่เรียกเก็บเงินได้ ซึ่งรวมถึงธุรกรรมเวลาและวัสดุและธุรกรรมราคาคงที่</span><span class="sxs-lookup"><span data-stu-id="21707-105">Dynamics 365 Project Operations supports various accounting options for billable projects that include time and material and fixed price transactions.</span></span>
