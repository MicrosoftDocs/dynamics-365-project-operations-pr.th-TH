# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="9cf26-103">ตั้งค่าคอนฟิกส่วนประกอบที่คิดค่าธรรมเนียมได้ของรายการใบเสนอราคาตามโครงการ</span><span class="sxs-lookup"><span data-stu-id="9cf26-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="9cf26-104">_**นำไปใช้กับ:** Project Operations สำหรับสถานการณ์ตามทรัพยากร/วัสดุที่ไม่ได้เก็บในคลัง_</span><span class="sxs-lookup"><span data-stu-id="9cf26-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="9cf26-105">รายการใบเสนอราคาตามโครงการสามารถมีส่วนประกอบที่รวมและส่วนประกอบที่คิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="9cf26-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="9cf26-106">ส่วนประกอบที่รวมจะเป็นไปตามการตั้งค่าวิธีการเรียกเก็บเงิน การแบ่งลูกค้า ขีดจำกัดที่ต้องไม่เกิน และความถี่ของใบแจ้งหนี้ ที่กำหนดไว้ในรายการใบเสนอราคาตามโครงการ</span><span class="sxs-lookup"><span data-stu-id="9cf26-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>
<span data-ttu-id="9cf26-107">ส่วนย่อยของส่วนประกอบที่รวมสามารถทำเครื่องหมายเป็นสามารถคิดค่าธรรมเนียมได้โดยใช้ **ชนิดการเรียกเก็บเงิน**</span><span class="sxs-lookup"><span data-stu-id="9cf26-107">A subset of the included components can be marked as chargeable by using **Billing Type**.</span></span> <span data-ttu-id="9cf26-108">คุณสามารถเลือกหนึ่งในตัวเลือกต่อไปนี้ในฟิลด์ **ชนิดการเรียกเก็บเงิน** ในบริบทของรายการใบเสนอราคา:</span><span class="sxs-lookup"><span data-stu-id="9cf26-108">You can select one of the following options in the **Billing Type** field in the context of a quote line:</span></span>

   - <span data-ttu-id="9cf26-109">คิดค่าธรรมเนียมได้</span><span class="sxs-lookup"><span data-stu-id="9cf26-109">Chargeable</span></span>
   - <span data-ttu-id="9cf26-110">คิดค่าธรรมเนียมไม่ได้</span><span class="sxs-lookup"><span data-stu-id="9cf26-110">Non-chargeable</span></span>

<span data-ttu-id="9cf26-111">สามารถกำหนดส่วนประกอบที่สามารถคิดค่าธรรมเนียมได้ตามบทบาทและประเภทธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="9cf26-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="9cf26-112">ความสามารถในการเรียกเก็บเงินที่กำหนดไว้ในบทบาทสำหรับรายการใบเสนอราคาโครงการ จะใช้กับคลาสธุรกรรม **เวลา** เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="9cf26-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="9cf26-113">หากรายการใบเสนอราคาโครงการมี **รวมเวลา** = **ไม่** แท็บ **บทบาทที่คิดค่าธรรมเนียมได้** จะไม่พร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="9cf26-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="9cf26-114">ความสามารถในการเรียกเก็บเงินที่กำหนดไว้ในประเภทธุรกรรมสำหรับรายการใบเสนอราคาโครงการ จะใช้กับคลาสธุรกรรม **ค่าใช้จ่าย** เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="9cf26-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="9cf26-115">หากรายการใบเสนอราคาโครงการมี **รวมค่าใช้จ่าย** = **ไม่** แท็บ **ประเภทที่คิดค่าธรรมเนียมได้** จะไม่พร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="9cf26-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
