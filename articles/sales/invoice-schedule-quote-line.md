# <a name="invoice-schedules-on-project-based-quote-lines"></a><span data-ttu-id="a040a-103">กำหนดการใบแจ้งหนี้ในรายการใบเสนอราคาตามโครงการ</span><span class="sxs-lookup"><span data-stu-id="a040a-103">Invoice schedules on project-based quote lines</span></span>

<span data-ttu-id="a040a-104">_**นำไปใช้กับ:** Project Operations สำหรับสถานการณ์ตามทรัพยากร/วัสดุที่ไม่ได้เก็บในคลัง การปรับใช้งานแบบ Lite - จัดการกับการออกใบแจ้งหนี้ชั่วคราว_</span><span class="sxs-lookup"><span data-stu-id="a040a-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>
