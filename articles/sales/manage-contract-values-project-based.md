# <a name="work-with-projectbased-contract-lines"></a>ทำงานกับรายละเอียดการให้บริการตามสัญญาตามโครงการ

รายละเอียดการให้บริการตามสัญญาตามโครงการใน Dynamics 365 Project Operations ออกแบบมาเพื่อเก็บค่าการประมาณการและข้อตกลงการเรียกเก็บเงินสำหรับองค์ประกอบเฉพาะของงานโครงการในการมีส่วนร่วม โครงสร้างของรายละเอียดการให้บริการตามสัญญาตามโครงการจะขยายสำหรับการประมาณการโครงการและสถานการณ์การเรียกเก็บเงินด้วยแนวคิดต่อไปนี้:

- วิธีการเรียกเก็บเงิน
- การแมปโครงการและงาน
- คลาสธุรกรรมที่รวมไว้
- วงเงินที่ต้องไม่เกิน
- การตั้งค่าการคิดค่าธรรมเนียม
- ประมาณการโดยใช้รายละเอียดการให้บริการตามสัญญา
- ลูกค้าในรายละเอียดการให้บริการตามสัญญา

ฟิลด์ต่อไปนี้รวมอยู่ในแท็บ **ทั่วไป** ของรายละเอียดการให้บริการตามสัญญาตามโครงการ ฟิลด์เหล่านี้ช่วยในการตั้งค่าพื้นฐานสำหรับรายละเอียด การประมาณการพื้นฐาน และการเตรียมการเรียกเก็บเงินสำหรับงานตามโครงการ

| ฟิลด์ | รายละเอียด | ผลกระทบขั้นปลาย |
| --- | --- | --- |
| **ชื่อ** | ชื่อของรายละเอียดการให้บริการตามสัญญาที่ระบุส่วนประกอบที่ไม่ต่อเนื่องของสัญญาที่อยู่ระหว่างการประมาณ สำหรับสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากค่าที่สอดคล้องกันของรายการใบเสนอราคาตามโครงการ | ค่าฟิลด์นี้ถูกคัดลอกไปยังรายการใบแจ้งหนี้โครงการที่สร้างขึ้นจากรายละเอียดการให้บริการตามสัญญานี้เมื่อสร้างใบแจ้งหนี้ |
| **วิธีการเรียกเก็บเงิน** | บนสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา นี่คือชุดตัวเลือกที่แสดงถึงรูปแบบการทำสัญญาหลักสองแบบที่สนับสนุนโดย Project Operations:</br>- **ราคาคงที่**</br>- **เวลาและวัสดุ** | ตามวิธีการเรียกเก็บเงินของรายละเอียดการให้บริการตามสัญญาที่อ้างอิง ธุรกรรมจริงจะได้รับการประมวลผล หากรายละเอียดการให้บริการตามสัญญาที่อ้างอิงตามจริงมีวิธีการเรียกเก็บเงินตามเวลาและวัสดุ ระบบจะสร้างเรกคอร์ดต้นทุนและยอดขายที่ยังไม่เรียกเก็บ หากรายละเอียดการให้บริการตามสัญญาที่อ้างอิงตามจริงมีวิธีการเรียกเก็บเงินแบบราคาคงที่ ระบบจะสร้างเฉพาะต้นทุนจริงเท่านั้น |
| **Project** | ใช้ฟิลด์นี้เพื่อระบุโครงการที่จะใช้ในการส่งมอบงานในการมีส่วนร่วมนี้ | ค่าในฟิลด์นี้ใช้ร่วมกับฟิลด์ **รวมงาน** และ **รวมคลาสธุรกรรม** เพื่อแก้ไขการอ้างอิงรายการสัญญาในเรกคอร์ดรายการจริงหรือรายการโดยประมาณ |
| **รวมเวลา** | ค่าสถานะระบุว่าธุรกรรมเวลาหรือต้นทุนแรงงานในโครงการที่เลือกรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าธุรกรรมเวลาหรือต้นทุนแรงงานจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่าฟิลด์นี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดจริงหรือรายการโดยประมาณ |
| **รวมค่าใช้จ่าย** | ค่าสถานะระบุว่าต้นทุนค่าใช้จ่ายในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าต้นทุนค่าใช้จ่ายจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่าฟิลด์นี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดจริงหรือรายการโดยประมาณ |
| **รวมค่าธรรมเนียม** | ค่าสถานะระบุว่าค่าธรรมเนียมในโครงการที่เลือกจะรวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้หรือไม่ ค่า **ไม่** ระบุว่าค่าธรรมเนียมจะไม่รวมอยู่ในรายละเอียดการให้บริการตามสัญญานี้ ค่า **ใช่** บ่งชี้ว่ารวม | ค่าฟิลด์นี้ใช้ร่วมกับโครงการเพื่อแก้ไขการอ้างอิงรายละเอียดการให้บริการตามสัญญาในเรกคอร์ดจริงหรือรายการโดยประมาณ |
| **จำนวนเงินตามสัญญา** | ในรายละเอียดการให้บริการตามสัญญาราคาคงที่ นี่คือมูลค่าที่ตกลงกันซึ่งจะออกใบแจ้งหนี้ให้กับลูกค้าสำหรับส่วนประกอบงานทั้งหมดที่เกี่ยวข้องกับรายละเอียดการให้บริการตามสัญญานี้ ในรายละเอียดการให้บริการตามสัญญาเวลาและวัสดุ จำนวนนี้คือค่าประมาณที่จะออกใบแจ้งหนี้ให้กับลูกค้าสำหรับส่วนประกอบงานทั้งหมดที่เกี่ยวข้องกับรายละเอียดการให้บริการตามสัญญานี้ บนสัญญาโครงการที่สร้างจากใบเสนอราคา ค่านี้จะถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา เมื่อรายละเอียดการให้บริการตามสัญญาตามโครงการมีรายละเอียดรายการ ฟิลด์นี้จะถูกล็อกสำหรับการแก้ไขและสรุปจากจำนวนเงินในรายละเอียดการให้บริการตามสัญญา | เมื่อรายละเอียดการให้บริการตามสัญญามีรายละเอียดของรายการ ค่านี้สามารถแก้ไขได้โดยการเปลี่ยนจำนวนเงินในรายละเอียดรายการ ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างจำนวนเงินก่อนหักภาษีสำหรับเหตุการณ์การเรียกเก็บเงินตามงวด |
| **ภาษีที่ประเมิน** | ผู้ใช้สามารถแก้ไขฟิลด์นี้เพื่อป้อนจำนวนภาษีโดยประมาณในรายละเอียดการให้บริการตามสัญญา เมื่อรายละเอียดการให้บริการตามสัญญาตามโครงการมีรายละเอียดรายการ ฟิลด์นี้จะถูกล็อกสำหรับการแก้ไขและสรุปจากจำนวนเงินภาษีในรายละเอียดการให้บริการตามสัญญา | เมื่อรายละเอียดการให้บริการตามสัญญามีรายละเอียดของรายการ ค่านี้สามารถแก้ไขได้โดยการเปลี่ยนจำนวนเงินภาษีในรายละเอียดรายการ ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างภาษีสำหรับเหตุการณ์การเรียกเก็บเงินตามงวด |
| **จำนวนเงินตามสัญญาหลังหักภาษี** | จำนวนเงินตามรายละเอียดการให้บริการตามสัญญาหลังหักภาษี ฟิลด์นี้เป็นแบบอ่านอย่างเดียวและคำนวณเป็น **จำนวนเงินตามสัญญา + ภาษี** | ในบรายละเอียดการให้บริการตามสัญญาราคาคงที่ ค่านี้จะใช้เพื่อสร้างเหตุการณ์การเรียกเก็บเงินตามงวด |
| **วงเงินที่ต้องไม่เกิน** | ผู้ใช้สามารถแก้ไขฟิลด์นี้และใช้ได้เฉพาะในรายละเอียดการให้บริการตามสัญญาตามโครงการที่มีวิธีการเรียกเก็บเงินเวลาและวัสดุ | ผู้ใช้สามารถแก้ไขฟิลด์นี้ เมื่อเวลาหรือค่าใช้จ่ายจริงอ้างถึรายละเอียดการให้บริการตามสัญญาสำหรับเวลาและวัสดุ จำนวนเงินตามจริงจะถูกประเมินเทียบกับไม่เกินขีดจำกัด ในรายละเอียดการให้บริการตามสัญญานี้หลังจากการทำบัญชีสำหรับจำนวนเงินที่ใช้จ่ายไปแล้วและยืนยันแล้ว |
| **งบประมาณของลูกค้า** | ค่านี้สามารถแก้ไขได้และถูกคัดลอกจากฟิลด์ที่สอดคล้องกันบนรายการใบเสนอราคา หากสัญญาสร้างจากใบเสนอราคา | ฟิลด์นี้ใช้สำหรับข้อมูลเท่านั้นและไม่มีความสำคัญดาวน์สตรีม |
