<span data-ttu-id="b3b0f-142">โหมดงานที่จัดกำหนดการด้วยตนเองไม่ใช้ชั่วโมงทำงาน ปฏิทินโครงการ หรือจำนวนของทรัพยากรที่กำหนดไว้ในงาน</span><span class="sxs-lookup"><span data-stu-id="b3b0f-142">Manually scheduled task mode doesn't honor the work hours, project calendar, or number of resources defined on the task.</span></span> <span data-ttu-id="b3b0f-143">กำหนดการงานเป็นไปตามการป้อนข้อมูลของผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="b3b0f-143">The task schedule is based on user input.</span></span> <span data-ttu-id="b3b0f-144">สำหรับงานดังกล่าว การจัดสรรความพยายามต่อหน่วยรอบระยะเวลาของมาตราส่วนเวลาที่เลือก ไม่มีปัจจัยข้อจำกัดใดๆ</span><span class="sxs-lookup"><span data-stu-id="b3b0f-144">For such tasks, the effort allocation per unit time period of the chosen timescale doesn't have any limiting factors.</span></span> <span data-ttu-id="b3b0f-145">ความพยายามรวมในงานจะถูกแบ่งเท่าๆกัน และถูกปันส่วนสำหรับแต่ละหน่วยรอบระยะเวลาบนมาตราส่วนเวลาที่เลือก</span><span class="sxs-lookup"><span data-stu-id="b3b0f-145">The total effort on the task is equally split and allocated for each unit time period on the chosen timescale.</span></span>  
  
<span data-ttu-id="b3b0f-146">ด้วยวิธีนี้ โหมดงานที่กำหนดไว้ในงานจะกำหนดกระจายของความพยายาม หรือการปันส่วนของความพยายามต่อหน่วยรอบระยะเวลาในการประเมินตามระยะเวลา</span><span class="sxs-lookup"><span data-stu-id="b3b0f-146">In this way, the task mode defined on the task determines the effort distribution or allocation of effort per unit time period in time-phased estimates.</span></span>  
  
## <a name="grouping-and-time-phasing-options"></a><span data-ttu-id="b3b0f-147">ตัวเลือกการจัดกลุ่ม และการแบ่งระยะเวลา</span><span class="sxs-lookup"><span data-stu-id="b3b0f-147">Grouping and time-phasing options</span></span>  
<span data-ttu-id="b3b0f-148">มุมมองนี้ช่วยให้คุณเข้าใจการกระจายของความพยายาม ต้นทุน และการประเมินการขายบนพื้นฐานต่อวัน สัปดาห์ เดือน หรือปี</span><span class="sxs-lookup"><span data-stu-id="b3b0f-148">This view helps you understand the distribution of the effort, cost, and sales estimates on a per day, week, month, or year basis.</span></span> <span data-ttu-id="b3b0f-149">ตัวเลือก จัดกลุ่มโดย อนุญาตให้การขยับข้อมูลการประเมินในมิติอื่นๆสองมิติ: ประเภทและทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="b3b0f-149">The Group By option allows pivoting the estimates data on two other dimensions: category and resource.</span></span> <span data-ttu-id="b3b0f-150">ในทั้งมุมมองกริดและมุมมองตามระยะเวลา คุณสามารถเลือกฟิลด์ที่จะแสดงขึ้นได้</span><span class="sxs-lookup"><span data-stu-id="b3b0f-150">On both the grid view and the time-phased view you can choose the fields to be displayed.</span></span> <span data-ttu-id="b3b0f-151">ผลรวมสำหรับแต่ละช่วงเวลาจะแสดงขึ้นที่ด้านล่าง เพื่อแสดงความพยายามที่มีการประเมินรวม ต้นทุน และการขายสำหรับวัน สัปดาห์ เดือน หรือปี</span><span class="sxs-lookup"><span data-stu-id="b3b0f-151">Totals for each of the time blocks are displayed at the bottom indicating the total estimated effort, cost, and sales for the day, week, month, or year.</span></span>  
  
<span data-ttu-id="b3b0f-152">ราคาต้นทุนและราคาขายที่เป็นค่าเริ่มต้นจะเป็นวันที่ที่มีผลบังคับใช้</span><span class="sxs-lookup"><span data-stu-id="b3b0f-152">The cost and sales price default is date effective.</span></span> <span data-ttu-id="b3b0f-153">เมื่ออัตราสำหรับบทบาทเปลี่ยนแปลง จะมองเห็นได้ง่ายมากขึ้นในมุมมองการแบ่งเวลา เมื่อดูข้อมูลประเมินที่ขยับบน 'ทรัพยากร' และแบ่งเวลาตามสัปดาห์</span><span class="sxs-lookup"><span data-stu-id="b3b0f-153">When the rates for the roles change, it will be more transparent in the time-phased view when viewing estimate data pivoted on ‘Resource’ and time-phased by week.</span></span>
