## <a name="differences-between-project-service-automation-and-sales"></a><span data-ttu-id="085a9-168">ความแตกต่างระหว่าง Project Service Automation และ Sales</span><span class="sxs-lookup"><span data-stu-id="085a9-168">Differences between Project Service Automation and Sales</span></span>
