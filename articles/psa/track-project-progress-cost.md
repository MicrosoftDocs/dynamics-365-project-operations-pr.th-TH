# <a name="track-project-progress-and-cost-project-service"></a><span data-ttu-id="34189-103">ติดตามความคืบหน้าของโครงการและต้นทุน (Project Service)</span><span class="sxs-lookup"><span data-stu-id="34189-103">Track project progress and cost (Project Service)</span></span>

[!INCLUDE[cc-applies-to-psa-app-1x-2x](../includes/cc-applies-to-psa-app-1x-2x.md)]

<span data-ttu-id="34189-104">ความต้องการสำหรับการติดตามความคืบหน้าเทียบกับโครงสร้างการแบ่งงานแตกต่างกันไปตามแผนภูมิแนวตั้งของอุตสาหกรรม</span><span class="sxs-lookup"><span data-stu-id="34189-104">The need for tracking progress against a work breakdown structure varies across industry verticals.</span></span> <span data-ttu-id="34189-105">บางอุตสาหกรรมทำได้ดีกว่าที่ระดับละเอียดมากของโครงสร้างการแบ่งงาน ในขณะที่บางอุตสาหกรรมอาจติดตามที่โครงสร้างการแบ่งงานระดับสูงขึ้น</span><span class="sxs-lookup"><span data-stu-id="34189-105">Some industries do it more diligently at a very granular level of the work breakdown structure, while others might track at a higher level of the work breakdown structure.</span></span>  
  
## <a name="effort-tracking-view"></a><span data-ttu-id="34189-106">มุมมองการติดตามความพยายามที่ใช้</span><span class="sxs-lookup"><span data-stu-id="34189-106">Effort tracking view</span></span>  
<span data-ttu-id="34189-107">มุมมองการติดตามความพยายามที่ใช้แสดงการติดตามของความคืบหน้าของงานในโครงสร้างการแบ่งงาน</span><span class="sxs-lookup"><span data-stu-id="34189-107">The effort tracking view shows the tracking of progress for tasks in the work breakdown structure.</span></span> <span data-ttu-id="34189-108">ซึ่งจะเปรียบเทียบจำนวนชั่วโมงของความพยายามจริงกับงานจนถึงปัจจุบันกับจำนวนชั่วโมงของความพยายามที่วางแผนไว้ที่จะใช้ไปกับงาน</span><span class="sxs-lookup"><span data-stu-id="34189-108">It compares the actual effort hours spent against a task till date to the planned effort hours on a task.</span></span>  
  
<span data-ttu-id="34189-109">บางครั้งคุณอาจจำเป็นต้องแก้ไขการประเมินเดิมของงาน</span><span class="sxs-lookup"><span data-stu-id="34189-109">Sometimes you might need to revise the original estimates on a task.</span></span> <span data-ttu-id="34189-110">งานอาจย้ายเร็วหรือช้ากว่าที่คาดไว้ตั้งแต่แรก เนื่องจากการเปลี่ยนแปลงภายในขอบเขตหรือมีบุคคลที่มีประสบการณ์น้อยกว่าที่วางแผนไว้ตั้งแต่แรก</span><span class="sxs-lookup"><span data-stu-id="34189-110">The task might be moving faster or slower than originally anticipated because of changes in scope or having less experienced people than originally planned for.</span></span> <span data-ttu-id="34189-111">เราไม่แนะนำให้เปลี่ยนเลขเส้นฐาน เนื่องจากเส้นฐานของโครงการเป็นแหล่งที่เผยแพร่สำหรับกำหนดการและการประเมินต้นทุนของโครงการที่ผู้เกี่ยวข้องทั้งหมดในโครงการได้ตกลง</span><span class="sxs-lookup"><span data-stu-id="34189-111">We don't recommend changing your baseline numbers, because the project baseline is the published source for the project’s schedule and cost estimates that all stakeholders on the project have agreed to.</span></span>
