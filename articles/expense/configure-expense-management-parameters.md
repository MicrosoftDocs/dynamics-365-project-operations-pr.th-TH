# <a name="configure-expense-management-parameters"></a>กำหนดค่าพารามิเตอร์การจัดการค่าใช้จ่าย

_**นำไปใช้กับ:** Project Operations สำหรับสถานการณ์ตามทรัพยากร/วัสดุที่ไม่ได้เก็บในคลัง_

หัวข้อนี้อธิบายถึงพารามิเตอร์ที่ควบคุมลักษณะการทำงานทั่วไปในการจัดการค่าใช้จ่าย

## <a name="general"></a>ทั่วไป

| เขตข้อมูล                                                    | รายละเอียด |
|----------------------------------------------------------|-------------|
| อัตรามาตรฐานของระยะไมล์                                 | ป้อนอัตราการชำระเงินคืนสำหรับค่าใช้จ่ายระยะไมล์ อัตรานี้คูณด้วยระยะไมล์ที่ป้อนสำหรับค่าใช้จ่ายเพื่อคำนวณจำนวนเงินที่จะได้รับคืนสำหรับค่าใช้จ่ายในการเดินทาง |
| ตรวจสอบวัตถุประสงค์ของค่าใช้จ่าย                                 | เปิดใช้ตัวเลือกนี้เพื่อจำกัดผู้ใช้ตามชุดของค่าที่มีอยู่ซึ่งกำหนดค่าไว้ในฟิลด์ **วัตถุประสงค์ของรายงานค่าใช้จ่าย** เมื่อสร้างรายงานค่าใช้จ่าย |
| ค่าใช้จ่ายส่วนตัวที่จ่ายโดย                                | เลือกผู้ที่รับผิดชอบในการชำระเงินธุรกรรมบัตรเครดิตใดๆ ที่จัดอยู่ในประเภทส่วนบุคคล |
| แสดงรายงานค่าใช้จ่ายทั้งหมดมื่อดูรายละเอียดย้อนกลับ               | เลือกตัวเลือกนี้เพื่อแสดงค่าใช้จ่ายทั้งหมดสำหรับรายงานค่าใช้จ่ายเมื่อมีการดูรายละเอียดของเอกสารต้นฉบับสำหรับใบสำคัญเฉพาะที่สร้างขึ้นเมื่อมีการลงรายการบัญชีรายงานค่าใช้จ่าย |
| การขออนุมัติการเดินทางล่วงหน้าเป็นสิ่งจำเป็น                 | เลือกตัวเลือกนี้เพื่อกำหนดให้ต้องส่งใบเบิกค่าเดินทางและได้รับการอนุมัติก่อนที่พนักงานจะสามารถส่งรายงานค่าใช้จ่ายได้ |
| อนุญาตให้แก้ไขการกระจายก่อนการลงรายการบัญชี            | เลือกว่าสามารถแก้ไขการกระจายของรายงานค่าใช้จ่ายก่อนที่จะลงรายการบัญชีได้หรือไม่ |
| ประเมินนโยบายการจัดการค่าใช้จ่าย                     | เลือกเวลาที่มีการประเมินค่าใช้จ่ายเพื่อพิจารณาว่ามีการละเมิดนโยบายค่าใช้จ่ายหรือไม่ ค่าใช้จ่ายสามารถประเมินสำหรับการละเมิดเมื่อป้อนและบันทึกรายการค่าใช้จ่าย หรือเมื่อมีการส่งค่าใช้จ่ายเพื่อขอการอนุมัติ กฎนโยบายสำหรับใบเสร็จรับเงินที่จำเป็นจะได้รับการประเมินเมื่อบันทึกรายการค่าใช้จ่าย |
| อนุญาตรายการค่าใช้จ่ายระหว่างบริษัท                         | เลือกว่าจะป้อนค่าใช้จ่ายสำหรับนิติบุคคลอื่นในรายงานค่าใช้จ่ายหรือไม่ |
| อนุญาตให้แก้ไขอัตราแลกเปลี่ยนสำหรับค่าใช้จ่ายบัตรเครดิต | เลือกตัวเลือกนี้เพื่ออนุญาตให้ผู้ใช้แก้ไขอัตราแลกเปลี่ยนสำหรับค่าใช้จ่ายบัตรเครดิตที่นำเข้า |
| ฟิลด์ลำดับชั้นหลายระดับที่จะแสดง                  | เลือกฟิลด์ผู้อนุมัติที่จะแสดงเมื่อชนิดการตั้งค่าชนิดการกำหนดเวิร์กโฟลว์เป็นลำดับชั้น และการเลือก **ลำดับชั้น** มีการกำหนดค่าให้ใช้ลำดับชั้นการอนุมัติ การอนุมัติหลายระดับของค่าใช้จ่าย เมื่อใช้ลำดับชั้นการอนุมัติหลายระดับสำหรับเวิร์กโฟลว์ ฟิลด์ที่เลือกจะแสดงในรายงานค่าใช้จ่ายและสามารถแก้ไขได้ |
| ป้อนหมายเลขบัตรเครดิตของพนักงาน                        | เลือกว่าจะป้อนและบันทึกหมายเลข 15 อักขระหรือ 16 อักขระในฟิลด์ **รหัสบัตร** บนเพจ **บัตรเครดิต** สำหรับพนักงาน |
| ตรวจสอบวัตถุประสงค์ของใบเบิกค่าเดินทาง                      | เปิดใช้ตัวเลือกนี้เพื่อจำกัดผู้ใช้ตามชุดของค่าที่มีอยู่ซึ่งกำหนดค่าไว้ในฟิลด์ **วัตถุประสงค์ของรายงานค่าใช้จ่าย** เมื่อสร้างใบเบิกค่าเดินทาง |

## <a name="financial"></a>การเงิน

| เขตข้อมูล                                                                                    | รายละเอียด |
|------------------------------------------------------------------------------------------|-------------|
| ชื่อสมุดรายวันประจำวันของบัญชีแยกประเภท                                                                | เลือกชื่อของสมุดรายวันบัญชีแยกประเภทที่มีการลงรายการบัญชีรายงานค่าใช้จ่ายที่อนุมัติ |
| เปิดใช้งานการหักคืนภาษีจากค่าใช้จ่าย                                                        | เลือกตัวเลือกนี้เพื่อเปิดใช้งานการหักคืนภาษีค่าใช้จ่ายสำหรับค่าใช้จ่ายที่มีสิทธิ์ ไม่สามารถเลือกตัวเลือกนี้ได้หากเปิดใช้กฎภาษีขายและภาษีนำเข้าของสหรัฐอเมริกา |
| ลงรายการบัญชีการเบิกเงินสดล่วงหน้าทันที                                                           | เลือกตัวเลือกนี้เพื่อลงรายการบัญชีการเบิกเงินสดล่วงหน้าที่ได้รับอนุมัติเมื่อกระบวนการชำระเงินและการโอนเสร็จสิ้น หากไม่ได้เลือกตัวเลือกนี้ กระบวนการชำระเงินและการโอนจะสร้างสมุดรายวันทั่วไปที่ยังไม่ได้ลงรายการบัญชี |
| แก้ไขวันที่การบัญชีระหว่างการลงรายการบัญชี                                                   | เลือกตัวเลือกนี้เพื่อปรับปรุงวันที่การบัญชีเมื่อมีการลงรายการบัญชีค่าใช้จ่าย หากรอบระยะเวลาปัจจุบันถูกระงับหรือถูกปิดแล้ว วันที่การบัญชีจะถูกกำหนดเป็นระยะเวลาที่เปิดถัดไป |
| อนุญาตให้จัดกลุ่มธุรกรรมตามบัญชีออฟเซ็ตที่ระบุในวิธีการชำระเงิน       | เลือกตัวเลือกนี้เพื่อสรุปธุรกรรมค่าใช้จ่ายสำหรับรายงานค่าใช้จ่าย โดยยึดตามบัญชีออฟเซ็ตที่ระบุไว้ในวิธีการชำระเงินสำหรับค่าใช้จ่าย |
| รวมภาษี                                                                             | เลือกตัวเลือกนี้เพื่อรวมภาษีขายในยอดเงินค่าใช้จ่ายตามค่าเริ่มต้น |
| ปลดภาระผูกพันในการปิดใบเบิกค่าเดินทาง                                      | เลือกตัวเลือกนี้เพื่อปลดเงินทุนที่มีภาระผูกพันเมื่อมีการปิดใบเบิกค่าที่ได้รับอนุมัติ |
| อนุญาตให้ส่งใบเบิกค่าเดินทางเมื่อมีการใช้จ่ายเกินงบประมาณสำหรับทะเบียนงบประมาณและงบประมาณโครงการ | เลือกตัวเลือกนี้เพื่ออนุญาตให้พนักงานส่งใบเบิกค่าเดินทางสำหรับการอนุมัติ แม้ว่าจะเกินงบประมาณที่อนุญาตซึ่งกำหนดไว้ในทะเบียนงบประมาณหรืองบประมาณที่อนุญาตสำหรับโครงการ |
| อนุญาตให้ส่งรายงานค่าใช้จ่ายเมื่อมีการใช้จ่ายเกินงบประมาณสำหรับทะเบียนงบประมาณและงบประมาณโครงการ     | เลือกตัวเลือกนี้เพื่ออนุญาตให้พนักงานส่งรายงานค่าใช้จ่ายสำหรับการอนุมัติ แม้ว่าจะเกินงบประมาณที่อนุญาตซึ่งกำหนดไว้ในทะเบียนงบประมาณหรืองบประมาณที่อนุญาตสำหรับโครงการ |
| อนุญาตให้มีการอนุมัติรายงานค่าใช้จ่ายเมื่อมีการใช้จ่ายเกินงบประมาณสำหรับทะเบียนงบประมาณเท่านั้น                 | เลือกตัวเลือกนี้เพื่ออนุญาตให้ผู้อนุมัติอนุมัติรายงานค่าใช้จ่ายที่เกินงบประมาณที่อนุญาตซึ่งกำหนดไว้ในทะเบียนงบประมาณ |

## <a name="per-diem"></a>เบี้ยเลี้ยง

| เขตข้อมูล                                 | รายละเอียด |
|---------------------------------------|-------------|
| ชั่วโมงขั้นต่ำสำหรับค่าเบี้ยเลี้ยงต่อวัน            | ป้อนจำนวนชั่วโมงขั้นต่ำเริ่มต้นที่พนักงานต้องทำงานในหนึ่งวันเพื่อให้มีสิทธิ์ได้รับค่าเบี้ยเลี้ยงต่อวันสำหรับค่าใช้จ่ายที่เกี่ยวข้องกับการเดินทาง ค่านี้ใช้เป็นค่าเริ่มต้นสำหรับฟิลด์ **ชั่วโมงขั้นต่ำ** สำหรับระดับอัตราค่าเบี้ยเลี้ยงต่อวัน |
| เปอร์เซ็นต์ค่าอาหาร                          | ป้อนเปอร์เซ็นต์เริ่มต้นของค่าเบี้ยเลี้ยงต่อวันสำหรับมื้ออาหารที่ใช้ในวันแรกและวันสุดท้ายของค่าใช้จ่ายที่เกี่ยวข้องกับการเดินทางเมื่อฟิลด์ **คำนวณการลดมื้ออาหารตาม** ตั้งค่าเป็น **ชนิดมื้ออาหารต่อวัน** หรือ **จำนวนมื้ออาหารต่อวัน** วันทำงานในวันแรกและวันสุดท้ายอาจสั้นกว่าวันทำงานมาตรฐาน ดังนั้น จำนวนเงินของค่าเบี้ยเลี้ยงต่อวันในวันนั้นๆ อาจแตกต่างจากจำนวนเงินมาตรฐาน หากตั้งค่าเปอร์เซ็นต์เป็น **0** (ศูนย์) การหักเงินสำหรับวันแรกและวันสุดท้ายจะเป็น 0.00 |
| เปอร์เซ็นต์ค่าโรงแรม                         | ป้อนเปอร์เซ็นต์เริ่มต้นของค่าเบี้ยเลี้ยงต่อวันสำหรับโรงแรมที่ใช้ในวันแรกและวันสุดท้ายของค่าใช้จ่ายที่เกี่ยวข้องกับการเดินทาง วันทำงานในวันแรกและวันสุดท้ายอาจสั้นกว่าวันทำงานมาตรฐาน ดังนั้น จำนวนเงินของค่าเบี้ยเลี้ยงต่อวันในวันนั้นๆ อาจแตกต่างจากจำนวนเงินมาตรฐาน หากตั้งค่าเปอร์เซ็นต์เป็น **0** (ศูนย์) การหักเงินสำหรับวันแรกและวันสุดท้ายจะเป็น 0.00 |
| เปอร์เซ็นต์อื่นๆ                         | ป้อนเปอร์เซ็นต์เริ่มต้นของค่าเบี้ยเลี้ยงต่อวันสำหรับค่าใช้จ่ายเบ็ดเตล็ดที่ใช้ในวันแรกและวันสุดท้ายของค่าใช้จ่ายที่เกี่ยวข้องกับการเดินทาง วันทำงานในวันแรกและวันสุดท้ายอาจสั้นกว่าวันทำงานมาตรฐาน ดังนั้น จำนวนเงินของค่าเบี้ยเลี้ยงต่อวันในวันนั้นๆ อาจแตกต่างจากจำนวนเงินมาตรฐาน หากตั้งค่าเปอร์เซ็นต์เป็น **0** (ศูนย์) การหักเงินสำหรับวันแรกและวันสุดท้ายจะเป็น 0.00 |
| การหักลดเปอร์เซ็นต์สำหรับอาหารเช้า | ป้อนจำนวนเงินที่หักลดของค่าเบี้ยเลี้ยงต่อวันสำหรับอาหารเช้า ตัวอย่างเช่น หากพนักงานได้รับอาหารเช้าฟรี คุณอาจต้องการหักลดจำนวนเงินค่าเบี้ยเลี้ยงต่อวันลง 10 เปอร์เซ็นต์ |
| การหักลดเปอร์เซ็นต์สำหรับอาหารกลางวัน     | ป้อนจำนวนเงินที่หักลดของค่าเบี้ยเลี้ยงต่อวันสำหรับอาหารกลางวัน ตัวอย่างเช่น หากพนักงานได้รับอาหารกลางวันฟรี คุณอาจต้องการหักลดจำนวนเงินค่าเบี้ยเลี้ยงต่อวันลง 15 เปอร์เซ็นต์ |
| การหักลดเปอร์เซ็นต์สำหรับอาหารเย็น    | ป้อนจำนวนเงินที่หักลดของค่าเบี้ยเลี้ยงต่อวันสำหรับอาหารเย็น ตัวอย่างเช่น หากพนักงานได้รับอาหารเย็นฟรี คุณอาจต้องการหักลดจำนวนเงินค่าเบี้ยเลี้ยงต่อวันลง 25 เปอร์เซ็นต์ |
| คำนวณการหักลดมื้ออาหารตาม           | ระบุว่าระบบควรคำนวณการหักลดมื้ออาหารจากค่าเบี้ยเลี้ยงต่อวันอย่างไร โดยเลือกว่าการหักลดนั้นขึ้นอยู่กับชนิดมื้ออาหารต่อการเดินทางหรือต่อวัน หรือตามจำนวนมื้ออาหารที่อนุญาตต่อวัน |
| การปัดเศษค่าเบี้ยเลี้ยงต่อวัน                     | เลือกชนิดของการปัดเศษที่ใช้สำหรับค่าใช้จ่ายของค่าเบี้ยเลี้ยงต่อวัน หากคุณเลือกการปัดเศษตามปกติ ค่าใช้จ่ายของค่าเบี้ยเลี้ยงต่อวันที่มีจำนวนเงิน 0.50 จะถูกปัดเศษขึ้นเป็น 1.00 และค่าใช้จ่ายของค่าเบี้ยเลี้ยงต่อวันใดๆ ที่มีจำนวนเงินน้อยกว่า 0.50 จะถูกปัดเศษลงเป็น 0.00 |
| การคำนวณค่าเบี้ยเลี้ยงต่อวันพื้นฐานตาม          | เลือกว่าจำนวนเงินของค่าเบี้ยเลี้ยงต่อวันจะขึ้นอยู่กับวันตามปฏิทินหรือช่วงเวลา 24 ชั่วโมง |

| เขตข้อมูล                          | รายละเอียด |

| คำแนะนำ                   | ป้อนคำแนะนำที่พนักงานต้องปฏิบัติตามเมื่อสร้างใบปะหน้าสำหรับแฟกซ์ที่ใช้ในการส่งใบเสร็จรับเงินที่เกี่ยวข้องกับรายงานค่าใช้จ่าย หากต้องการป้อนข้อความเฉพาะภาษาที่จะแสดงตามภาษาของผู้ใช้ ให้เลือก **คำแปล** |

## <a name="anti-corruption"></a>การป้องกันการทุจริต

| เขตข้อมูล                                 | รายละเอียด |
|---------------------------------------|-------------|
| แสดงเอกสารรับรองเพื่อการป้องกันการทุจริต   | เลือกตัวเลือกนี้เพื่อแสดงข้อความเพื่อการป้องกันการทุจริตเมื่อสร้างรายงานค่าใช้จ่าย จากนั้นสามารถเปิดใช้งานประเภทค่าใช้จ่ายเฉพาะซึ่งจะต้องมีการเลือกการรับรองเพื่อการป้องกันการทุจริตในรายงานค่าใช้จ่าย ตัวอย่างเช่น ประเภทของขวัญที่เกี่ยวข้องกับค่าใช้จ่ายของข้าราชการอาจต้องการให้พนักงานยืนยันว่าค่าใช้จ่ายเป็นไปตามนโยบายของบริษัทที่เกี่ยวข้องกับข้าราชการ |
| ข้อความเพื่อการป้องกันการทุจริตสำหรับผู้ส่ง | ป้อนข้อความที่ควรแสดงต่อพนักงานที่กำลังสร้างรายงานค่าใช้จ่าย หากต้องการป้อนข้อความเฉพาะภาษาที่จะแสดงตามภาษาของผู้ใช้ ให้เลือก **คำแปล** |
| ข้อความเพื่อการป้องกันการทุจริตสำหรับผู้อนุมัติ  | ป้อนข้อความที่ควรแสดงต่อผู้อนุมัติเมื่อมีการสร้างรายงานค่าใช้จ่าย หากต้องการป้อนข้อความเฉพาะภาษาที่จะแสดงตามภาษาของผู้ใช้ ให้เลือก **คำแปล** |
