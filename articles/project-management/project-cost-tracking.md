## <a name="labor-cost-tracking-view"></a><span data-ttu-id="33572-108">มุมมองการติดตามต้นทุนแรงงาน</span><span class="sxs-lookup"><span data-stu-id="33572-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="33572-109">บนเพจ **โครงการ** บนแท็บ **การติดตาม** คุณสามารถเลือก **การติดตาม** > **ต้นทุน** เพื่อเปิดมุมมอง **การติดตามต้นทุน** และดูความคืบหน้าของการใช้จ่ายค่าแรงในแต่ละงานในแผนโครงการ</span><span class="sxs-lookup"><span data-stu-id="33572-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="33572-110">มุมมองนี้ยังเปรียบเทียบต้นทุนแรงงานจริงที่ใช้ในงานกับต้นทุนแรงงานตามแผนของงานด้วย</span><span class="sxs-lookup"><span data-stu-id="33572-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="33572-111">Project Operations ใช้สูตรต่อไปนี้เพื่อคำนวณเมตริกต้นทุนแรงงาน:</span><span class="sxs-lookup"><span data-stu-id="33572-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="33572-112">**ต้นทุนตามแผน**: ต้นทุนโดยประมาณของการกำหนดทรัพยากรทั้งหมดในงานโหนดปลายสุดแต่ละงาน</span><span class="sxs-lookup"><span data-stu-id="33572-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="33572-113">**ต้นทุนจริง**: ผลรวมของต้นทุนที่เกิดขึ้นจริงสำหรับเวลาที่บันทึกไว้ในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="33572-114">**เปอร์เซ็นต์การใช้ต้นทุน**: ต้นทุนจริง ÷ ประมาณการต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="33572-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>

- <span data-ttu-id="33572-116">**ต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์**: ต้นทุนที่เหลือ - ต้นทุนจริง</span><span class="sxs-lookup"><span data-stu-id="33572-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="33572-117">**ผลต่างต้นทุน**: ต้นทุนตามแผน - ประมาณการต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="33572-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="33572-118">แต่ละงานจะแสดงการวางแผนของผลต่างต้นทุนในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="33572-119">หากประมาณการต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์มากกว่าต้นทุนตามแผน งานนั้นจะคาดว่าเกินงบประมาณ</span><span class="sxs-lookup"><span data-stu-id="33572-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="33572-120">หากประมาณการต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์น้อยกว่าต้นทุนตามแผน งานนั้นจะคาดว่าเสร็จสิ้นโดยน้อยกว่างบประมาณ</span><span class="sxs-lookup"><span data-stu-id="33572-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="33572-133">การวางแผนต้นทุนในงานโหนดปลายสุดใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-133">Reprojecting costs on leaf node tasks</span></span>

<span data-ttu-id="33572-134">ต้นทุนแรงงานในงานโหนดปลายสุดไม่สามารถวางแผนใหม่ได้โดยตรงบนแท็บ **การติดตาม** ในเพจ **โครงการ**</span><span class="sxs-lookup"><span data-stu-id="33572-134">Labor costs on a leaf node task can't be directly reprojected on the **Tracking** tab on the **Project page**.</span></span> <span data-ttu-id="33572-135">อย่างไรก็ตาม คุณสามารถใช้มุมมอง **การติดตามกำลังคน** เพื่อวางแผนกำลังคนที่เหลือในงานใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-135">However, you can use the **Effort Tracking** view to reproject the remaining effort on a task.</span></span> <span data-ttu-id="33572-136">ซึ่งทำให้เกิดการคำนวณต้นทุนที่เหลือของงานใหม่อีกครั้ง</span><span class="sxs-lookup"><span data-stu-id="33572-136">This causes a recalculation of the remaining cost on the task.</span></span> <span data-ttu-id="33572-137">ต่อไปนี้เป็นคำอธิบายวิธีการทำงานนี้</span><span class="sxs-lookup"><span data-stu-id="33572-137">The following is a description of how this works.</span></span>

1. <span data-ttu-id="33572-138">ผู้จัดการโครงการสามารถวางแผนกำลังคนในงานใหม่ได้โดยการปรับปรุงค่าเริ่มต้นในฟิลด์ **กำลังคนที่เหลือ** ด้วยประมาณการใหม่ของการดำเนินงานที่เหลือในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-138">A project manager can reproject effort on tasks by updating the default value in the **Remaining Effort** field with a new estimate of the remaining effort on the task.</span></span> <span data-ttu-id="33572-139">การวางแผนใหม่ทำให้เกิดการคำนวณใหม่ของประมาณการกำลังคนเมื่อดำเนินงานเสร็จสมบูรณ์ (EAC) ของงาน และเปอร์เซ็นต์ความคืบหน้า และผลต่างความพยายามที่คาดการณ์ไว้ในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-139">Reprojecting causes a recalculation of the task's effort estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="33572-140">EAC, ประมาณการเมื่อดำเนินงานเสร็จสมบูรณ์ (ETC) และเปอร์เซ็นต์ความคืบหน้าในงานสรุปจะถูกคำนวณอีกครั้ง และสร้างการวางแผนใหม่ของผลต่างกำลังคน</span><span class="sxs-lookup"><span data-stu-id="33572-140">The EAC, estimate to complete (ETC), and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
2. <span data-ttu-id="33572-141">ระบบจะคำนวณต้นทุนที่เหลือในมุมมอง **การติดตามต้นทุน** ขึ้นอยู่กับมูลค่าใหม่สำหรับกำลังคนที่เหลือในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-141">Based on the new value for the remaining effort on the task, the system calculates the remaining cost on the **Cost Tracking** view.</span></span> <span data-ttu-id="33572-142">ในการคำนวณต้นทุนที่เหลือโดยพิจารณาจากกำลังคนที่เหลืออยู่ ระบบจะคำนวณต้นทุนเฉลี่ยของการดำเนินงานหนึ่งชั่วโมงในงานนั้นตามต้นทุนตามแผนหรือการดำเนินงานตามแผน</span><span class="sxs-lookup"><span data-stu-id="33572-142">To calculate the remaining cost based on the remaining effort, the system first calculates the average cost of one hour of effort on the task as planned cost or planned effort.</span></span> <span data-ttu-id="33572-143">ต้นทุนตามแผนคือผลรวมของต้นทุนของการกำหนดทรัพยากรทั้งหมดในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-143">The planned cost is the sum of the cost of all resource assignments on the task.</span></span> <span data-ttu-id="33572-144">ต้นทุนเฉลี่ยต่อชั่วโมงจะใช้ในการคำนวณต้นทุนที่เหลือสำหรับกำลังคนที่เหลืออยู่ที่เพิ่งวางแผนในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-144">The average cost per hour is used to compute the remaining cost for the newly projected remaining effort on the task.</span></span>
3. <span data-ttu-id="33572-145">ต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์และเปอร์เซ็นต์การใช้ต้นทุนในงานโหนดปลายสุดจะมีการคำนวณใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-145">The cost at complete and cost consumption percentage on the leaf-node task are re-calculated.</span></span>
4. <span data-ttu-id="33572-146">มูลค่าของต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์ของงานสรุปรวมถึงโหนดรากจะมีการคำนวณใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-146">The cost at complete value of the summary tasks all the way to the root node are recalculated.</span></span>

## <a name="reprojecting-costs-on-summary-tasks"></a><span data-ttu-id="33572-147">การวางแผนต้นทุนในงานสรุปใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-147">Reprojecting costs on summary tasks</span></span>

<span data-ttu-id="33572-148">คุณสามารถวางแผนต้นทุนแรงงานใหม่สำหรับงานสรุปหรืองานคอนเทนเนอร์</span><span class="sxs-lookup"><span data-stu-id="33572-148">You can reproject labor costs on summary tasks or container tasks.</span></span> <span data-ttu-id="33572-149">อย่างไรก็ตาม คุณไม่สามารถวางแผนต้นทุนแรงงานในงานโครงการสรุปใหม่ได้บนแท็บ **การติดตาม** ในเพจ **โครงการ**</span><span class="sxs-lookup"><span data-stu-id="33572-149">However, you can't directly reproject labor cost on a summary project task on the **Tracking** tab on the **Project** page.</span></span> <span data-ttu-id="33572-150">เช่นเดียวกับงานโหนดปลายสุด การวางแผนงานสรุปและงานคอนเทนเนอร์ใหม่สามารถทำได้โดยใช้มุมมอง **การติดตามกำลังคน**</span><span class="sxs-lookup"><span data-stu-id="33572-150">Similar to leaf node tasks, reprojecting summary and container tasks can be done using the **Effort Tracking** view.</span></span> <span data-ttu-id="33572-151">ในมุมมองนี้ คุณสามารถวางแผนกำลังคนที่เหลือในงานสรุปใหม่ได้ ซึ่งทำให้เกิดการคำนวณต้นทุนที่เหลือในงานสรุปใหม่</span><span class="sxs-lookup"><span data-stu-id="33572-151">In this view, you can reproject the remaining effort on a summary task causing a recalculation of the remaining cost on the summary task.</span></span> <span data-ttu-id="33572-152">ต่อไปนี้เป็นคำอธิบายวิธีการทำงานนี้</span><span class="sxs-lookup"><span data-stu-id="33572-152">The following is a description of how this works.</span></span>

1. <span data-ttu-id="33572-153">ผู้จัดการโครงการสามารถวางแผนกำลังคนในงานสรุปใหม่ได้โดยการปรับปรุง ค่าเริ่มต้นของการดำเนินงานที่เหลือด้วยประมาณการใหม่ของงาน</span><span class="sxs-lookup"><span data-stu-id="33572-153">A project manager can reproject effort on summary tasks by updating the default value of the remaining effort with a new estimate on the task.</span></span> <span data-ttu-id="33572-154">การปรับปรุงนี้ทำให้เกิดการคำนวณใหม่ของประมาณการเมื่อดำเนินงานเสร็จสมบูรณ์ (EAC) ของงานสรุป และเปอร์เซ็นต์ความคืบหน้า และผลต่างของกำลังคนที่คาดการณ์ไว้ในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-154">This update causes a recalculation of the summary task's estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="33572-155">EAC, ETC, และเปอร์เซ็นต์ความคืบหน้าในงานสรุปจะถูกคำนวณอีกครั้ง และสร้างการวางแผนใหม่ของผลต่างกำลังคน</span><span class="sxs-lookup"><span data-stu-id="33572-155">The EAC, ETC, and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
2. <span data-ttu-id="33572-156">ระบบจะคำนวณต้นทุนที่เหลือในมุมมอง **การติดตามต้นทุน** ขึ้นอยู่กับมูลค่าใหม่สำหรับกำลังคนที่เหลือในงานสรุป</span><span class="sxs-lookup"><span data-stu-id="33572-156">Based on the new value for remaining effort on the summary task, the system calculates the remaining cost on the **Cost Tracking** view.</span></span> <span data-ttu-id="33572-157">ในการคำนวณต้นทุนที่เหลือโดยพิจารณาจากกำลังคนที่เหลืออยู่ ระบบจะคำนวณต้นทุนเฉลี่ยของการดำเนินงานหนึ่งชั่วโมงในงานสรุปตามต้นทุนตามแผนหรือการดำเนินงานตามแผน</span><span class="sxs-lookup"><span data-stu-id="33572-157">To calculate the remaining cost based on the remaining effort, the system first calculates the average cost of one hour of effort on the summary task as planned cost or planned effort.</span></span> <span data-ttu-id="33572-158">ต้นทุนเฉลี่ยต่อชั่วโมงจะใช้ในการคำนวณต้นทุนสำหรับกำลังคนที่เหลืออยู่ที่เพิ่งวางแผนในงานสรุป</span><span class="sxs-lookup"><span data-stu-id="33572-158">The average cost per hour is used to calculate the cost of the newly projected remaining effort on the summary task.</span></span>

4. <span data-ttu-id="33572-160">ต้นทุนเมื่อดำเนินงานเสร็จสมบูรณ์ใหม่จะกระจายลงไปยังงานย่อยในสัดส่วนเดียวกันกับต้นทุนตามแผนเดิมที่อยู่ในงาน</span><span class="sxs-lookup"><span data-stu-id="33572-160">The new cost at complete is distributed down to the child tasks in the same proportion as the original planned cost was on the task.</span></span>

<span data-ttu-id="33572-164">ฟิลด์ **ประสิทธิภาพต้นทุน** สามารถตั้งค่าได้จากข้อมูลการติดตาม</span><span class="sxs-lookup"><span data-stu-id="33572-164">The **Cost performance** field can be set from the tracking data.</span></span> <span data-ttu-id="33572-165">เมื่อผลต่างต้นทุนสำหรับโหนดรากในมุมมอง **การติดตามค่าใช้จ่าย** เป็นลบ คุณสามารถตั้งค่าฟิลด์นี้เป็น **ต่ำกว่างบประมาณ**</span><span class="sxs-lookup"><span data-stu-id="33572-165">When the cost variance for the root node in the **Cost tracking** view is negative, you can set this field to **Under Budget**.</span></span> <span data-ttu-id="33572-166">เมื่อผลต่างต้นทุนสำหรับโหนดรากเป็นค่าบวก คุณสามารถตั้งค่าเป็น **เกินงบประมาณ**</span><span class="sxs-lookup"><span data-stu-id="33572-166">When the cost variance for the root node is positive, you can set the value to **Over Budget**.</span></span>
