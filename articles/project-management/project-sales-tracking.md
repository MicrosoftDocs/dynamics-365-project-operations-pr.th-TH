4. <span data-ttu-id="2cce5-142">มูลค่าของรายได้เมื่อดำเนินงานเสร็จสมบูรณ์ของงานสรุปรวมถึงโหนดรากจะมีการคำนวณใหม่</span><span class="sxs-lookup"><span data-stu-id="2cce5-142">The revenue at complete value of the summary tasks all the way to the root node are recalculated.</span></span>

## <a name="reprojecting-revenues-on-summary-tasks"></a><span data-ttu-id="2cce5-143">การวางแผนรายได้ใหม่ในงานสรุป</span><span class="sxs-lookup"><span data-stu-id="2cce5-143">Reprojecting revenues on summary tasks</span></span>

<span data-ttu-id="2cce5-144">คุณสามารถวางแผนรายได้ต่อแรงงานใหม่สำหรับงานสรุปหรืองานคอนเทนเนอร์</span><span class="sxs-lookup"><span data-stu-id="2cce5-144">You can reproject labor revenue on summary tasks or container tasks.</span></span> <span data-ttu-id="2cce5-145">อย่างไรก็ตาม คุณไม่สามารถวางแผนรายได้ต่อแรงงานใหม่ในงานโครงการสรุปได้บนแท็บ **การติดตาม** ในเพจ **โครงการ**</span><span class="sxs-lookup"><span data-stu-id="2cce5-145">However, you can't directly reproject labor revenues on a summary project task on the **Tracking** tab on the **Project** page.</span></span> <span data-ttu-id="2cce5-146">เช่นเดียวกับงานโหนดปลายสุด การวางแผนงานสรุปและงานคอนเทนเนอร์ใหม่สามารถทำได้โดยใช้มุมมอง **การติดตามกำลังคน**</span><span class="sxs-lookup"><span data-stu-id="2cce5-146">Similar to leaf node tasks, reprojecting summary and container tasks can be done using the **Effort Tracking** view.</span></span> <span data-ttu-id="2cce5-147">ในมุมมองนี้ คุณสามารถวางแผนกำลังคนที่เหลือในงานสรุปใหม่ได้ ซึ่งทำให้เกิดการคำนวณรายได้ที่เหลือในงานสรุปใหม่</span><span class="sxs-lookup"><span data-stu-id="2cce5-147">In this view, you can reproject the remaining effort on a summary task causing a recalculation of the remaining revenue on the summary task.</span></span> <span data-ttu-id="2cce5-148">ต่อไปนี้เป็นคำอธิบายวิธีการทำงานนี้</span><span class="sxs-lookup"><span data-stu-id="2cce5-148">The following is a description of how this works.</span></span>

1. <span data-ttu-id="2cce5-149">ผู้จัดการโครงการสามารถวางแผนกำลังคนในงานใหม่ได้โดยการปรับปรุงค่าเริ่มต้นในฟิลด์ **กำลังคนที่เหลือ** ด้วยประมาณการใหม่ของ **กำลังคนที่เหลือ** ในงาน</span><span class="sxs-lookup"><span data-stu-id="2cce5-149">A project manager can reproject effort on tasks by updating the default value in the **Remaining Effort** field with a new estimate of the **Remaining Effort** on the task.</span></span> <span data-ttu-id="2cce5-150">การวางแผนใหม่ทำให้เกิดการคำนวณใหม่ประมาณการเมื่อดำเนินงานเสร็จสมบูรณ์ (EAC) ของงาน และเปอร์เซ็นต์ความคืบหน้า และผลต่างกำลังคนที่คาดการณ์ไว้ในงาน</span><span class="sxs-lookup"><span data-stu-id="2cce5-150">Reprojecting causes a recalculation of the task's estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="2cce5-151">EAC, ETC, และเปอร์เซ็นต์ความคืบหน้าในงานสรุปจะถูกคำนวณอีกครั้ง และสร้างการวางแผนใหม่ของผลต่างกำลังคน</span><span class="sxs-lookup"><span data-stu-id="2cce5-151">The EAC, ETC, and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
2. <span data-ttu-id="2cce5-152">ขึ้นอยู่กับมูลค่าใหม่สำหรับฟิลด์ **กำลังคนที่เหลือ** ในงาน ระบบจะคำนวณรายได้ที่เหลือในมุมมอง **การติดตามรายได้**</span><span class="sxs-lookup"><span data-stu-id="2cce5-152">Based on the new value in the **Remaining effort** field on the task, the system calculates the remaining revenue in the **Revenue Tracking** view.</span></span> <span data-ttu-id="2cce5-153">ในการคำนวณรายได้ที่เหลือโดยพิจารณาจากกำลังคนที่เหลืออยู่ ระบบจะคำนวณรายได้เฉลี่ยของกำลังคนหนึ่งชั่วโมงในงานตามรายได้ตามแผนหรือกำลังคนตามแผน</span><span class="sxs-lookup"><span data-stu-id="2cce5-153">To calculate the remaining revenue based on remaining effort, the system first calculates the average revenue of one hour of effort on the task as planned revenue or planned effort.</span></span> <span data-ttu-id="2cce5-154">รายได้ตามแผนคือผลรวมของรายได้ของการกำหนดทรัพยากรทั้งหมดในงาน</span><span class="sxs-lookup"><span data-stu-id="2cce5-154">The planned revenue is the sum of the revenue of all resource assignments on the task.</span></span> <span data-ttu-id="2cce5-155">รายได้เฉลี่ยต่อชั่วโมงนี้จะใช้ในการคำนวณรายได้สำหรับกำลังคนที่เหลืออยู่ที่เพิ่งวางแผนในงาน</span><span class="sxs-lookup"><span data-stu-id="2cce5-155">This average revenue per hour  is then used to calculate the revenue of the newly projected remaining effort on the task.</span></span>
3. <span data-ttu-id="2cce5-156">รายได้โดยประมาณเมื่อดำเนินงานเสร็จสมบูรณ์และเปอร์เซ็นต์การใช้รายได้ในงานสรุปจะมีการคำนวณใหม่</span><span class="sxs-lookup"><span data-stu-id="2cce5-156">The estimated revenue at complete and revenue consumption percentages on the summary task are re-calculated.</span></span>

6. <span data-ttu-id="2cce5-161">มูลค่าของรายได้เมื่อดำเนินงานเสร็จสมบูรณ์ของงานสรุปรวมถึงโหนดรากจะมีการคำนวณใหม่</span><span class="sxs-lookup"><span data-stu-id="2cce5-161">The revenue at complete value of the summary tasks all the way to the root node are recalculated.</span></span>
