# <a name="per-diem-rules"></a><span data-ttu-id="38f0e-103">กฎค่าเบี้ยเลี้ยงต่อวัน</span><span class="sxs-lookup"><span data-stu-id="38f0e-103">Per diem rules</span></span>

<span data-ttu-id="38f0e-104">*ค่าเบี้ยเลี้ยงต่อวัน* คือเบี้ยเลี้ยงที่จ่ายให้กับผู้ปฏิบัติงานที่กำลังเดินทางไปทำงาน</span><span class="sxs-lookup"><span data-stu-id="38f0e-104">A *per diem* is an allowance that is paid to a worker who is traveling for work.</span></span> <span data-ttu-id="38f0e-105">ในการจัดการค่าใช้จ่าย คุณสามารถสร้างกฎค่าเบี้ยเลี้ยงต่อวันสำหรับสถานการณ์การเดินทางต่างๆ</span><span class="sxs-lookup"><span data-stu-id="38f0e-105">In Expense management, you can create per diem rules for various travel situations.</span></span> <span data-ttu-id="38f0e-106">ค่าเบี้ยเลี้ยงต่อวันอาจขึ้นอยู่กับช่วงเวลาของปี สถานที่เดินทาง หรือทั้งสองอย่าง</span><span class="sxs-lookup"><span data-stu-id="38f0e-106">Per diem rates can be based on the time of year, the travel location, or both.</span></span> <span data-ttu-id="38f0e-107">เมื่อคุณสร้างกฎค่าเบี้ยเลี้ยงต่อวัน คุณสามารถระบุว่าเปอร์เซ็นต์ของอัตราค่าเบี้ยเลี้ยงต่อวันจะถูกระงับหากผู้ปฏิบัติงานได้รับอาหารหรือบริการฟรี</span><span class="sxs-lookup"><span data-stu-id="38f0e-107">When you create a per diem rule, you can specify that a percentage of the per diem rate will be withheld if a worker receives complimentary meals or services.</span></span> <span data-ttu-id="38f0e-108">คุณยังสามารถกำหนดจำนวนชั่วโมงขั้นต่ำและสูงสุดที่อัตราค่าเบี้ยเลี้ยงต่อวันใช้กับการเดินทางของผู้ปฏิบัติงานได้</span><span class="sxs-lookup"><span data-stu-id="38f0e-108">You can also set a minimum and maximum number of hours that the per diem rate can apply to a worker's travel.</span></span>

<span data-ttu-id="38f0e-109">การคำนวณยอดรวมค่าเบี้ยเลี้ยงต่อวันจะขึ้นอยู่กับวันที่เริ่มต้นและวันที่สิ้นสุดที่คุณสามารถกำหนดได้ในหน้า **ค่าเบี้ยเลี้ยงต่อวัน**</span><span class="sxs-lookup"><span data-stu-id="38f0e-109">The calculation of a total per diem amount is based on the start dates and end dates that you can set on the **Per diems** page.</span></span>
